## CALCULATOR

Welcome to Calculator Web App! This repository hosts an interactive and visually appealing calculator built with HTML, CSS, and JavaScript. 

## FEATURES
- Effortless calculations: Perform basic arithmetic operations such as addition, subtraction, multiplication, and division with just a few clicks.
- Responsive design: The calculator seamlessly adapts to different screen sizes, allowing for a smooth user experience on any device.
- Sophisticated user interface: A sleek and intuitive interface ensures that using the calculator is a delightful experience.
- Keyboard support: For those who prefer using their keyboard, this calculator also responds to key inputs, making it even more convenient.

## HOW TO USE
1. Clone this repository to your local machine.
  
   git clone https://github.com/jinosabu1999/calculator.git
   
2. Open index.html in your preferred web browser.
3. Voila! You have reached the Calculator Web App and are ready to perform calculations with ease.
